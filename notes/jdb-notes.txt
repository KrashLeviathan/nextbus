----------------------------------------
    Running the Java Debugger (jdb)
----------------------------------------

First, compile the java class.

  $ javac -g <filename>.java

Start JDB.

  $ jdb <filename>

Set breakpoints before running

  > stop in <method name>  [  Example: stop in filename.main(java.lang.String[])  ]

or...

  > stop at <line number>

Run the program

  > run



----------------------------------------
         COMMANDS WITHIN JDB
----------------------------------------

> list
    Shows lines above and below the current line.

> next
    Runs the current line and moves to the next.

> step
    Steps into the current line.

> step up
    Steps out of a method.

> stepi
    Steps through byte code indices (individual sub-steps in a line of code)

> where
    Shows the call stack.

> print <instance>
    Used to print instances and variables. [  Example: > print obj.x  ]

> dump <instance>
    Used to print full details of an instance.

> locals
    Shows the values of all local variables.

> watch <instance>.<variable>
    Used to stop the code when a variable value changes.

> unwatch <instance>.<variable>
    Removes the watch.

> monitor <action>
    Used to define what actions to take when hitting a breakpoint.
    [  Example: > monitor list  ]
    Use without parameters to show all actions monitored.

> unmonitor <action>
    Undoes a monitor.

> cont
    Continues the code until another break point or watch.

> clear
    Shows all breakpoints.

> clear <breakpoint>
    Clears a breakpoint.

> classes
    Show all classes loaded.

> class <classname>
    Show details about a particular class.

> methods <classname>
    Show all methods in that class.

> fields <classname>
    Show all instance variable fields in that class.

> trace methods
    Stops code execution on entering/exiting a method.

> untrace methods
    Returns back to normal execution.



----------------------------------------
             And finally...
----------------------------------------

Run your program to see if it worked!

  $ java <filename>
